# Dashboard Matrix Exchange

Community-shared Dashboard Matrix dashboard layouts and screenshots.

## Structure

```text
layouts/
  callsign/
    YYYYMMDDTHHMMSSZ-callsign-layout-name.json
    YYYYMMDDTHHMMSSZ-callsign-layout-name.png
```

Exports should be generated by Dashboard Matrix's layout export API or
`scripts/export-layout.py`. Pull requests are validated automatically.

Never include passwords, GitHub tokens, unrelated private application data, or private contact
information in a shared layout.
