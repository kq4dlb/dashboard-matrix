# Changelog

## 0.1.0-beta

- Established Dashboard Matrix as an independent dashboard product.
- Added a first-run wizard and Blank, Amateur Radio, and Home Lab templates.
- Added layout import analysis and conflict strategies.
- Added Admin layout export, screenshot preview, and GitHub publishing.
- Added a complete Dashboard Matrix Exchange repository template.
- Added stable, beta, and nightly release-channel checks.
- Added per-source proxy diagnostics.
- Introduced the general map-provider SDK and separated DX Cluster integration.
- Introduced theme packages and the Matrix Light reference theme.
- Added explicit plugin permissions and environment-variable secret mappings.
- Added Windows PyInstaller and Raspberry Pi install/image workflows.
- Standardized product environment variables under `DASHBOARD_MATRIX_*`.
